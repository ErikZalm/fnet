The FNET is Dual TCP/IPv4/IPv6 stack (under GNU LGPLv3 license) for building embedded communication software on 32bit MCUs.

The stack provides following protocols and services:
  • Supported Platforms:
    • ARM Cortex-M4, Kinetis:
      • MK60N512 (TWR-K60N512).
      • MK70FN1M (TWR-K70F120M).
    • ColdFire:
      • Modelo - MCF54418 (TWR-MCF5441x).
      • Reindeer - MCF5282 (M5282EVB).
      • Kirin2 - MCF52235 (M52235EVB).
      • Kirin3 - MCF52259 (M52259EVB, M52259DEMO, TWR-MCF52259).
      • Lasko - MCF51CN128 (TWR-MCF51CN).
    • Qorivva Power Architecture:
      • Fado - MPC5668G (MPC5668GKIT).
  • Supported Compilers:
    • ARM:
      • Keil uVisionv v4.7 (MK60N512).
    • IAR:
      • Embedded Workbench for ARM v6.5 (MK60N512).
    • Freescale:
      • CodeWarrior Development Studio for Microcontrollers v10.4 (MK60N512, MK70FN1M, MCF52259).
      • CodeWarrior for ColdFire v7.2.2 (MCF5282, MCF52235, MCF52259).
      • CodeWarrior for Microcontrollers v6.3 (MCF51CN128).
      • CodeWarrior Development Studio for MPC55xx/MPC56xx v2.10 (MPC5668G).
  • Bare-metal TCP/IP stack.
  • Non-blocking Socket layer API (STREAM, DGRAM, RAW).
  • Core Protocols:
    • TCP.
    • UDP.
    • IPv4, ICMPv4, IGMP, ARP.
    • IPv6, ICMPv6, ND.
  • Network Interfaces:
    • Ethernet.
    • Loopback.
  • Services:
    • HTTP 1.0/0.9 Server:
      • GET, POST requests.
      • CGI, SSI.
      • Basic Access Authentication.
    • TELNET Server.
    • DNS Client.
    • DHCP Client.
    • TFTP Server/Client.
    • PING Service.
    • ROM File System.
    • Flash Memory Driver.
    • Command Line Shell.
    • Serial Driver.
  • Tools:
    • GUI File System Generation Tool.
    • GUI Throughput Benchmark Tool.
  • Applications:
    • TFTP Bootloader.
    • Shell Application.
    • Throughput Benchmark Application.
